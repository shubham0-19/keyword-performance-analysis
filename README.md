# Keyword Performance Analysis

## Project Overview
This project focuses on understanding website traffic behavior and evaluating Google AdWords performance.
The workflow includes data cleaning and restructuring in Excel, exploratory data analysis using Python, and interactive visualization using Power BI dashboards.
The objective is to convert raw marketing data into meaningful insights that support data-driven decision-making.

The goal is to identify:
- High traffic cost keywords
- Keywords with high CPC
- Keywords with maximum search volume
- Overall keyword performance trends

## Tools Used
- Power BI
- Excel
- Python

## Methods
- Data cleaning and preprocessing in Excel
- Data restructuring and validation
- Exploratory data analysis
- Data visualization
