# Keyword Performance Dashboard (Power BI)

## Project Overview
This project analyzes Google Ads keyword performance data using
Exploratory Data Analysis (EDA) and a Power BI dashboard.

The goal is to identify:
- High traffic cost keywords
- Keywords with high CPC
- Keywords with maximum search volume
- Overall keyword performance trends

## Tools Used
- Power BI
- Excel
- Python (for EDA)

## Key Insights
- PMP Certification keyword contributes the highest traffic cost
- High CPC keywords do not always have the highest search volume
- Search volume shows fluctuation over time
